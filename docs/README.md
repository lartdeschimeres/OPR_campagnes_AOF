# Campagne Dark Fantasy - Les Royaumes Déchus

Une campagne narrative pour **Age of Fantasy (OnePageRules)**, avec :
- Des **territoires à explorer** (citadelles, forêts, marais…).
- Des **dilemmes** à résoudre (choix moraux, pactes, trahisons).
- Un système de **sauvegarde/import en HTML**.

## Comment utiliser ?
1. Ouvre [`index.html`](https://[ton-username].github.io/campagne-dark-fantasy-opr/) dans ton navigateur.
2. Clique sur un territoire pour interagir.
3. Utilise les boutons **"Sauvegarder"** et **"Importer"** pour gérer tes parties.

## Arborescence

campagne-dark-fantasy-opr/
├── index.html          # Page principale
├── icones/             # Icônes des territoires
└── sauvegardes/        # Parties sauvegardées (HTML)
